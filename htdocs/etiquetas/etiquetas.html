Mostrar 102 Etiquetas HTML

  1. Etiqueta html

    Es el elemento raíz y define todo el documento HTML. Tiene una etiqueta de inicio html y una etiqueta de fin. Todos los demás elementos de la estructura HTML deben ser recogidos dentro de estas etiquetas.


  2. Etiqueta !DOCTYPE html

    Indica al navegador que el documento está basado en el estándar HTML5

  3. Etiqueta head

    Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. El resto de etiquetas de metadatos, irán recogidas dentro de las etiquetas de apertura y cierre del head. Importante explicar que estos metadatos del documento, es información para el navegador y no contenido que será visible en la página web.


  4. Etiqueta title

    Etiqueta usada para definir el título de la página web.


  5. Etiqueta link

    Se usa para enlazar recursos externos al documento HTML. El ejemplo más común son las hojas de estilos CSS..


  6. Etiqueta meta

    Etiqueta usada para definir otros metadatos que no se pueden definir con una etiqueta HTML especifica. Por ejemplo para definir el autor del sitio, o la descripción del mismo.


  7. Etiqueta style

    Etiquetas usadas para introducir código CSS en línea, es decir, en el propio documento HTML.


  8. Etiqueta body

    Todo lo que quieras mostrar en la página web debe ir recogido dentro de las etiquetas de apertura y cierre de body. Este contenido será el que se muestre en la web.


  9. Etiqueta nav

    Usadas para definir el contenido que será la sección de navegación de la web.


  10. Etiqueta main

    Se usa para definir el contenido principal del documento. Solamente puede existir uno por documento.


  11. Etiqueta seccion

    Define una sección del documento


  12. Etiqueta article

    Define contenido independiente de la web.


  13. Etiqueta aside

    Dentro de estas etiquetas suele alojarse el contenido adicional de la web. Suele ser contenido relacionado con la web pero de poca importancia


  14. h1,h2,h3,h4,h5,h6.

    Son etiquetas HTML muy importantes, ya que son usadas para jerarquizar el contenido de la web. Las etiquetas se usan para explicar brevemente el contenido que irá a continuación.


  15. Etiqueta header

    Se usan para definir la cabecera la página web. Suele contener el logotipo, menú de navegación, etc.


  16. Etiqueta footer

    Usadas para definir el pie de página.


  17. Etiqueta p

    Etiqueta usada para escribir párrafos de texto.


  18. Etiqueta embed

    Crea un contenedor para una aplicación externa.


  19. Etiqueta hr

    Etiqueta utilizada para «romper» entre dos secciones de una web. Usada comúnmente como separador.


  20. Etiqueta Pre

    Usada para pegar texto manteniendo el pre formato propio del texto.


  21. Etiqueta Blockquote

    Se usan para indicar que el contenido es texto citado


  22. Etiqueta ol

    Etiquetas para crear una lista ordenada


  23. Etiqueta ul

    Etiquetas para crear una lista des-ordenada


  24. Etiqueta li

    Etiquetas que recogen el contenido de un elemento de una lista, sea ordenada o no.


  25. Etiqueta dl

    Usada para crear una lista de definiciones.


  26. Etiqueta dt

    Representa un término definido por la siguiente etiqueta dd.


  27. Etiqueta dd

    Se usa para definir los términos listados antes que él.


  28. Etiqueta figure

    Indica una figura ilustrada como parte del documento HTML5.


  29. Etiqueta figcaption

    Utilizada para definir la leyenda de una figura.


  30. Etiqueta a

    Etiqueta utilizada para crear hiperenlaces en el documento HTML


  31. Etiqueta strong

    Etiqueta para definir una palabra o conjunto de ellas como importantes. Tiene una fuerte importancia en el SEO de la página.


  32. Etiqueta small

    Utilizada para dejar un comentario aparte, del tipo una nota de derechos de autoría, u otros textos que no son esenciales para la comprensión del documento.


  33. Etiqueta cite

    Para indicar el título de una obra


  34. Etiqueta sub

    Etiquetas utilizadas para representar un subíndice


  35. Etiqueta sup

    Etiquetas utilizadas para representar un superíndice


  36. Etiqueta mark

    Usada para resaltar texto


  37. Etiqueta span

    Etiqueta HTML sin ningún significado específico. Se usa conjuntamente con los atributos «class» o «id» para atribuirle ciertas características.


  38. Etiqueta br

    Etiqueta utilizada para crear un salto de línea


  39. Etiqueta img

    Etiqueta para «pintar» una imagen en la página web.


  40. Etiqueta iframe

    Es una etiqueta que sirve para anidar otro documento HTML dentro del documento principal.


  41. Etiqueta i

    Muestra el texto en itálica.


  42. Etiqueta object

    Utilizada llamar a un recurso externo de la web. Este recurso será tratado como una imagen, o un recurso externo para ser procesado por un plugin.


  43. Etiqueta video

    Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.


  44. Etiqueta audio

    Usada para cargar un archivo de audio o stream de audio.


  45. Etiqueta source

    Permite a autores especificar recursos multimedia alternativos para las etiquetas de video o audio.


  46. Etiqueta svg

    Se usa para llamar a una imagen vectorizada.


  47. Etiqueta table

    Etiquetas de apertura y cierre de una tabla. El resto de etiquetas de la tabla han de ir siempre recogidas entre estas dos etiquetas.


  48. Etiqueta caption

    Usada para indicar el título de la tabla.


  49. Etiqueta colgroup

    Etiqueta utilizada para agrupar dos o más columnas de una tabla.


  50. Etiqueta tbody

    Usada para describir los datos concretos de una tabla.


  51. Etiqueta thead

    Indica el bloque de filas que describen las etiquetas de las columnas de la tabla.


  52. Etiqueta tfoot

    Indica los bloques de filas que describen los resúmenes, o datos totales de una columna de una tabla.


  53. Etiqueta tr

    Se usa para indicar una fila de celdas de una tabla.


  54. Etiqueta td

    Usada para definir una celda de una tabla.


  55. Etiqueta th

    Etiqueta que se usa para definir el encabezado de una celda


  56. Etiqueta form

    Etiqueta de apertura y cierre de un formulario de una página web. El resto de etiquetas de formulario deben ir siempre recogidas entre estas etiquetas de apertura y cierre de formulario.


  57. Etiqueta fieldset

    Etiqueta que representa un conjunto o agrupación de elementos de un formulario. «Pinta» un recuadro alrededor de las etiquetas que estén contenidas dentro del fieldset


  58. Etiqueta legend

    Etiqueta ligada a fieldset. Indica el título del fieldset


  59. Etiqueta label

    Se usa para definir el nombre o título de un control del formulario.


  60. Etiqueta input

    Pinta un campo de introducción de datos para el usuario. Es de las principales etiquetas de un formulario.


  61. Etiqueta button

    Etiqueta utilizada para representar un botón en el formulario.


  62. Etiqueta select

    Input que permite una selección entre un conjunto de opciones.


  63. Etiqueta option

    Etiqueta ligada a select. Permite añadir diferentes opciones al select


  64. Etiqueta textarea

    Añade un campo al usuario para que pueda introducir texto en unas líneas máximas de texto que el desarrollador puede definir.


  65. Etiqueta ins

    Indica que el texto a sido insertado en lugar de otro.


  66. Etiqueta kbd

    Indica texto introducido por medio del teclado.


  67. Etiqueta keygen

    Encriptar datos de un formulario.


  68. Etiqueta menuitem

    Define un elemento de un menu creado con la etiqueta menu


  69. Etiqueta meter

    Define un elemento de un menu creado con la etiqueta menu


  70. Etiqueta noframes

    Muestra un contenido alterno en caso de que los frames no sean soportados.


  71. Etiqueta noscript

    Muestra un contenido altero en caso de que un script no sea soportado.


  72. Etiqueta optgroup

    Agrupa opciones relacionadas en una lista desplegable.


  73. Etiqueta param

    Define paramentros de un objeto de la etiqueta object.


  74. Etiqueta pre

    Etiqueta para escribir texto pre-formateado.


  75. Etiqueta progress

    Etiqueta para insertar una barra prograsiva.


  76. Etiqueta q

    Encierra un segmento de texto entre comillas.


  77. Etiqueta s

    Tacha texto con una linea horizontal.

    png

  78. Etiqueta samp

    Indica que el texto es un ejemplo de salida de un programa de computadora.


  79. Etiqueta script

    Etiqueta para insertar un script


  80. Etiqueta strike

    Indica invalidez a un fragmento de texto tachandolo con una línea horizontal.


  81. Etiqueta abbr

    Indicar una abreviación o acrónimo.


  82. Etiqueta acronym

    Indicar un acrónimo.


  83. Etiqueta address

    Etiqueta para incluir información de contacto.

  84. Etiqueta applet

    Insertar un applet en la página html.

  85. Etiqueta area

    Define un area de una imágen en donde se puede hacer clic.

  86. Etiqueta article

    Etiqueta semántica de HTML5 que indica la hubicación de un artículo

  87. Etiqueta bdi

    Aisla parte de un texto que podría estar en otro idioma o formato.

  88. Etiqueta bdo

    Cambia la dirección del texto.

  89. Etiqueta big

    Aumenta el tamaño del texto.

  90. Etiqueta canvas

    Etiqueta para definir un contenedor o lienzo para crear gráficos.

  91. Etiqueta center

    Centra el contenido.

  92. Etiqueta code

    Indica que el texto es código fuente de algún lenguaje informático.

  93. Etiqueta col

    Especifíca propiedades de atributos de las columnas de una tabla.

  94. Etiqueta colgroup

    Define un grupo de columnas de una tabla para darles formato.

  95. Etiqueta datalist

    Crea una lista de opciónes para una caja de texto input.

  96. Etiqueta details

    Especifica detalles que el usuario puede desplegar u ocultar.

  97. Etiqueta dfn

    Marca un término o definición.

  98. Etiqueta dialog

    Mostrar una caja de diálogo.

  99. Etiqueta dir

    Define una lista de directorios.

  100. Etiqueta div

    Crea un bloque.

  101. Etiqueta em

    Enfatizar parte del texto.

  102. Etiqueta time

    Indica una hora o fecha.
